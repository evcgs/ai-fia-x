## 二、MemOS 配置

确保 `memos-cloud-openclaw-plugin` 配置中开启多Agent模式：

```json
"multiAgentMode": true
```

每个Agent会自动使用独立的MemOS App ID：
- problemagent: `openclaw-problemagent`
- knowledgeagent: `openclaw-knowledgeagent`
- hypothesisagent: `openclaw-hypothesisagent`
- solutionagent: `openclaw-solutionagent`
- evaluationagent: `openclaw-evaluationagent`
- executionagent: `openclaw-executionagent`
- learningagent: `openclaw-learningagent`

## 三、验证配置

配置完成后重启网关，执行：
```bash
openclaw status
```
应该看到 **accounts 16/16** (或对应数量) 全部OK，说明配置正确。
