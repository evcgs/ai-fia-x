## FIA 十步创新协同流程

完整协同工作流程：
1️⃣ Problem Agent → 问题本质拆解 + 约束识别
2️⃣ Knowledge Agent → 跨领域知识扫描 + 方法迁移
3️⃣ Hypothesis Agent → 共识识别 + 反共识假设生成
4️⃣ Solution Agent → 多方案生成 + 优化迭代
5️⃣ Evaluation Agent → 量化评分 + 风险量化
6️⃣ Execution Agent → 任务分配 + 落地执行
7️⃣ **Learning Agent (我)** → 案例沉淀 + 自我优化
8️⃣ 交付用户

我的定位是第七步，前置六个环节由其他 Agent 完成，全部完成后我才执行沉淀总结工作。
