# FIA-X 多智能体创新架构

## 技能说明

FIA-X是系统化AI创新方法论，基于三大支柱：
- **First-principle** - 第一性原理问题拆解
- **Interdisciplinary** - 跨学科知识交叉迁移
- **Anti-consensus** - 反共识创新思维

通过七个专业智能体专业化分工，让AI真正产生突破性创新。

## 功能特性

- 七个专业智能体，每个专注一个创新环节
- 四道质量闸门，提前发现问题
- 完全符合OIA六步闭环执行标准
- 分层设计，OIA管协同，FIA管专业
- 每个智能体独立记忆存储，支持跨Agent检索
- 支持OpenClaw一键安装

## 安装

```bash
openclaw install evcgs/ai-fia-x
```

安装后按照 `docs/配置示例.md` 添加七个Agent到你的 `openclaw.json`，配置飞书App ID，重启网关即可使用。

## 要求

- OpenClaw >= 2026.3.0
- memos-cloud-openclaw-plugin >= 0.1.9
- 七个独立飞书应用（获取App ID和App Secret）
